# 僵尸猎人游戏

一个有趣的射击生存游戏，支持微信小游戏平台。

## 📁 项目结构

```
├── zombie-hunter-levels/    # H5闯关版
│   └── index.html           # 完整游戏代码
│
├── zombie-hunter-game/      # 微信小游戏版
│   ├── game.js              # 游戏主逻辑
│   ├── game.json            # 游戏配置
│   └── project.config.json  # 项目配置
│
└── zombie-hunter-game.zip   # 小游戏打包
```

## 🎮 游戏特性

- 6个关卡闯关模式
- 自动射击系统
- 升级技能强化
- 多种僵尸类型
- 炸弹技能
- 进度自动保存

## 🚀 快速开始

### H5版本
直接用浏览器打开 `zombie-hunter-levels/index.html`

### 微信小游戏版
1. 下载 [微信开发者工具](https://developers.weixin.qq.com/minigame/dev/devtools/download.html)
2. 导入 `zombie-hunter-game` 文件夹
3. 测试并发布

## 📱 跨设备开发

### 电脑端
1. 安装 [Git](https://git-scm.com/)
2. 克隆仓库：
   ```bash
   git clone https://github.com/你的用户名/zombie-hunter.git
   ```

### 手机端
1. 安装 [Termux](https://termux.com/) (Android) 或使用 Git客户端App
2. 克隆仓库进行开发
